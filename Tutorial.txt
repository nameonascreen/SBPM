1. Move "Steam.cfg" into C:\Program Files (x86)\Steam
/*
This will stop steam from ever updating when you start the client. This is required to keep the theme forever.
To Update steam just move this file out of the Steam directory. Doing so will remove the theme and you will need to reinstall it.
*/

2. Move "SBPM" Folder into C:\Program Files (x86)\Steam\skins

3. Run "steam.vbs"
//This will install the resource folder containing everything to do with the theme.
